## 🧐 About <a name = "about"> Boothcamp MySkill</a>

Boothcamp yang sangat menarik dan cocok untuk pemula jika ingin mempelajari tentang web development. Dalam 24 sesi diajarkan bagaimana mempelajari beberapa bahasa program seperti HTML,CSS dan Java Script dsb. Dari boothcamp ini saya telah berhasil membuat project individu MyPortofolio

## 🏁 Getting Started <a name = "getting_started"></a>

Setelah mempelajari HTML dan CSS saya belajar membuat project individu dengan membuat portofolio sendiri. Project ini saya buat dengan melakukan desain portofolio terlebih dahulu dengan wireframe dan figma.
